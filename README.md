# sueltas-app

Interactive dashboard for exploring typographic similarity across a corpus of historical books (BNE *sueltas* collection). Built with **Plotly Dash**.

## Quick start

```bash
git clone https://github.com/<owner>/sueltas-app.git
cd sueltas-app

# Download the image archive (~10 MB) — see "Image data" below
mkdir -p data/images
tar xzf images_cache.tar.gz -C data/images

conda create -n dash python=3.11 -y && conda activate dash
pip install -r requirements.txt
python scripts/preflight.py          # verify files & build caches
python launch_dashboard.py           # → http://localhost:8050
```

### Docker

```bash
git clone https://github.com/<owner>/sueltas-app.git
cd sueltas-app
# place images_cache.tar.gz in the repo root
docker build -t sueltas-app .
docker run -d -p 8050:8050 sueltas-app   # or: docker compose up -d
```

## Data

The repo includes all numerical matrices and metadata in `data/` (~2.8 MB). Only the letter images require a separate download.

### Tracked files (`data/`)

| File | Description |
|------|-------------|
| `n1hat_it_matrix_ordered.npy` | Italic similarity matrix (books × books) |
| `n1hat_rm_matrix_ordered.npy` | Roman similarity matrix |

| `impr_names_dashboard_ordered.npy` | Printer names per book |
| `symbs_dashboard.npy` | Symbol / character labels |
| `w_rm_matrix_ordered.npy` | Weight matrix — roman |
| `w_it_matrix_ordered.npy` | Weight matrix — italic |

| `n1hat_linkage_desc.npy` | Pre-computed linkage (desc) |
| `n1hat_linkage_desc_dist.npy` | Pre-computed linkage (desc dist) |

## Preflight check

```bash
python scripts/preflight.py              # full check + generate caches
python scripts/preflight.py --skip-umap  # skip slow UMAP computation
```

Verifies required files, image directories, and generates edge/UMAP caches. Exit codes: `0` = OK, `1` = missing required files, `2` = cache generation failed.

> Details: [`docs/FILE_DEPENDENCIES.md`](docs/FILE_DEPENDENCIES.md)

## Docker details

The `Dockerfile` bakes matrices (via `COPY`) and images (via `ADD images_cache.tar.gz`) into a single image. The `.dockerignore` excludes `data/images/`, `scripts/`, `docs/`, `.git/`, etc.

To rebuild after changing images:

```bash
python scripts/preflight.py                  # regenerate edge caches
tar czf images_cache.tar.gz -C data/images . # recreate archive
docker build -t sueltas-app .
```

For code-only or matrix-only changes, just `docker build`.

## Configuration

All parameters live in [`config.yaml`](config.yaml):

| Section | Controls |
|---------|----------|
| `server` | Port, host, debug, URL prefix |
| `data` | Paths to `.npy` files |
| `images` | Cache directory, `loading_strategy` (`preload` / `lazy`) |
| `network` | `top_k` edges, `n_bins`, default slider values |
| `umap` | `n_neighbors`, `min_dist` |
| `dendrogram` | Default cut level, truncation |
| `cache` | Figure/edge cache size limits |

## How it works

1. **`launch_dashboard.py`** reads `config.yaml` and creates a `BookSimilarityDashboard`.
2. **Network graph** — top-k most similar book pairs as edges; UMAP-positioned nodes; opacity-binned edges.
3. **Heatmap** — full similarity matrix ordered by hierarchical clustering.
4. **Letter comparison** — click two books to compare extracted letter images side-by-side.
5. **Dendrogram** — cut the linkage tree at a chosen level to reveal printer-coloured groups.
